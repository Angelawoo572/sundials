Dear Radu,

there is a japanese project called LiS standing for Large iterative
solvers. It is a collection of both sparse solvers and
preconditioners. I think that it is a public domain software without
licences and such things. It is intended for serial and parallel
implementation (using OpenMP or MPI). The IP with much more
information is http://ssi.is.s.u-tokyo.ac.jp/lis/index_en.html.

Vaggelis Voyiatzis
