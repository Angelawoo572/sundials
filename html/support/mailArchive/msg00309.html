Alan Hindmarsh <alanh@llnl.gov> wrote:

Hello Raj,

You wrote:
> Basically I want to have some flux calculations and check that
> source volume > CalculatedFlux*(t-ti)

I am a little concerned about this last comment. What is ti?

If ti is a past time that you have set externally to the solver, then
that calculation is fine. You would have to have saved a volume value
at ti in order to get the appropriate volume for this check.

But if by ti you mean the last internal t value that the solver
stepped to, this calculation is more complicated. You can get that ti
by way of CVodeGetCurrentTime and CVodeGetLastStep (ti = tcur-hlast).
But I don't see how you can calculate the appropriate source volume
that accumulated over the time interval (ti, t), a! t least in normal
output mode (itask=CV_NORMAL). If instead you are running in one-step
mode, and saving these values at every step, this would be easy.

-Alan Hindmarsh
