On Nov 28, 2006, at 7:36 PM, Aaron Collier wrote:

Dear Matthew Lanctot,

Yes, SUNDIALS includes generic Fortran-C interface (FCMIX) modules for the KINSOL, IDA and CVODE solvers. All you need to do is set the F77 environment variable to the appropriate compiler so the build system can determine the correct name mangling scheme. For example, since the Lahey/Fujitsu compiler is currently in my executable search path, I could configure SUNDIALS by issuing the following commands from a shell prompt:

  > cd build_dir
  > ../configure CC=gcc F77=lf95

You do not need to do anything special to use a Fortran 90/95 compiler. Just set F77 to the compiler you want to use.

Respectfully,
Aaron Collier
