Hi there,

I've created the sundials CMakeLists.txt files for the CMake cross build 
system. While they are so far only usable for the serial solvers and the 
Fortran/C mix is not included, they work really well for unix/linux/windows and 
particularly support compilers on the windows plattform natively (Cygwin is no 
longer required).
You can also create Visual Studio Project files with CMake. So for the various 
Windows compilers this should be an easy solution to get sundials compiled on 
this plattform.

You can find the necessary files and a tutorial on my webpage:

http://www.bauklimatik-dresden.de/privat/nicolai/html/en/sundials_cmake.html

This is the first version of the cmake build files, and they will probably be 
improved in the future. If you have suggestions or problems, please mail me.
