Vogel, Ernest F - Eastman wrote: 
For constraints, the KINSOL documentation says:

If constraints[i] is
0.0 then no constraint is imposed on ui.
1.0 then ui will be constrained to be ui > 0.0.
-1.0 then ui will be constrained to be ui < 0.0.
2.0 then ui will be constrained to be ui >= 0.0.
-2.0 then ui will be constrained to be ui <= 0.0.

From both looking at the KINSOL code and testing with an example, the 
documentation above is backwards regarding ui having a value of zero.

If 1.0 or -1.0, ui can be zero.  If 2.0 or -2.0, ui cannot be zero.

Ernie Vogel
