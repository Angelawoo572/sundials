libsundials_cvodes.lib(cvodes_io.obj) : warning LNK4075: ignoring 
'/EDITANDCONTINUE' due to '/INCREMENTAL:NO' specification
libsundials_cvodes.lib(cvodes_io.obj) : fatal error LNK1313: ijw/native module 
detected; cannot link with pure modules


This means that some part of your program, typically a GUI based on .Net is compiled with pure "Common Language Runtime support" 
enabled, while your libraries are not. "Pure" is part of the managed extensions that Microsoft introduced in there newest VC and 
"ensures", that you write nice and error free code (is there ever such thing?). For all libraries and programs in your project 
group, check the setting : "Configuration properties" -> "Common Language Runtime support"  and make sure you have that 
set to "No common language runtime support".

If you require the common language runtime support aka managed extensions or .net, you 
may try to set the CLRs option in the sundials libs to "Pure MSIL CLR".
