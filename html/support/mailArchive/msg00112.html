Hello Hang Su,

In general, the half-bandwidths (or any input parameters for any of
the linear solvers) affect only the performance of the linear solver
specified, but not the accuracy of the computed solution to the ODE or
DAE system itself.  That is because of the various convergence tests
for the linear and nonlinear algebraic system solution, and the local
error test for the ODE/DAE solution.  If bad inputs are given, the
linear solver will perform poorly (inefficiently), and the solver may
have to give up as a result, but the solution returned up to that
point should be accurate (in relation to the tolerances supplied).
The number of convergence failures (linear and nonlinear) is indeed a
marker for such a bad choice of inputs.  However, you may want to use
a choice that has a high linear convergence failure rate, if the run
still succeeds and has a lower overall cost than other choices.

-Alan H.
