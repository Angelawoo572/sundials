Building SUNDIALS/CVODE on Windows

From Henning Schmidt <henning@hschmidt.de>
Date Tue, 24 Jan 2006 12:07:06 +0100 
Hi,

I am a new user of the Sundials suite. Two weeks ago I 
first had a serious look at the CVODE solver to see if I could 
use it for speeding up the simulations in a MATLAB toolbox I am 
working on, which actually turned out to work VERY nicely.

Being a Windows user and not having written C programs for a couple 
of years, I initially had a bit of trouble building the needed libraries.
After some hours of trial and error it turned out to be very easy. 

In the following I am going to share my experiences, which hopefully will be
helpful for others who have trouble getting sundials suites built under
Windows. All downloads where done in mid January 2006 (so I do not need to
specify version numbers :)

Maybe I should mention that I am an electrical engineer and not a trained
software person. So some people might find my approach a bit cruel ... but
it is simple and it works just fine.

Unsuccessful try: Building SUNDIALS/CVODE libraries using Cygwin
================================================================
I downloaded and installed Cygwin. Then I downloaded the Sundials/CVODE
package and unzipped it in my Cygwin folder. The "configure" script ran
nicely and without any error messages or warnings. However, the "make"
command just returned an error message I couldn't interpret. Since I didn't
intend to make it work under Cygwin only, I did not spend time on finding
the problem. 

Working try: Building SUNDIALS/CVODE libraries using Visual Studio Express
==========================================================================
Visual Studio Express is Microsofts free C++ compiler, which can be found
here: http://msdn.microsoft.com/vstudio/express/visualc/. I believe the
program is very nice, however, it is also very complex in the way that 
it has tons of settings and it takes a while to really find the correct
settings (or even the menus where to change these settings). So here I will
describe how to use it from the command line (MUCH simpler than using the
IDE).

The following procedure will build a single library file for the CVODE IVP
solver. At the end this library is used to generate a first example
executable.

NOTE: The procedure described below should also work fine with other
ISO/ANSI C compilers and appropriate linkers.

1) After installation of the Visual Studio Express C++ you need to add 
   the path to the executables in the system path. You do that in the 
   standard Windows way using by "Start->Settings->Control Panel->System-> 
   Advanced->Environment Variables->System Variables->Path->Edit": Add the 
   following two entries:       
   C:\Program Files\Microsoft Visual Studio 8\VC\bin; 
   C:\Program Files\Microsoft Visual Studio 8\Common7\IDE
2) Download the Sundials/CVODE package and unzip to a desired location     
   (in this example I will use C:\ as this folder). 
3) Create the folders "C:\sundials\include" and C:\sundials\source"
4) Copy the contents of "C:\sundials\CVODE\include", 
   "C:\sundials\shared\include" into the new folder "C:\sundials\include".
   Also copy the .h files in the "C:\sundials\nvec_sec" (nvec_par) folder   
   into the C:\sundials\include folder.
5) Copy the contents of "C:\sundials\CVODE\source" and   
   "C:\sundials\shared\source" into the new folder "C:\sundials\source".
   Also copy the .c files in the "C:\sundials\nvec_ser" 
   ("C:\sundials\nvec_par") folder into the C:\sundials\source folder.
5) Create a new file "C:\sundials\include\sundials_config.h" with the  
   following content (just two lines): 
                #define SUNDIALS_DOUBLE_PRECISION
                #define SUNDIALS_USE_GENERIC_MATH
6) Start the command line application in windows
7) Change into the "C:\sundials\source" folder
8) Compilation to object code, type (one line):

cl /I "C:\sundials\include" 
   /I "C:\Program Files\Microsoft Visual Studio 8\VC\include" /c *.c

9) Binding the object files together to a static library file   

lib /OUT:sundials.lib *.obj

The library file is created! Now we will create a first executable example
based on it:

10) Compiling a first example:
        a) Create a new folder "C:\sundials\example"
        b) copy the "C:\sundials\cvode\examples_ser\cvbx.c" file into the
new 
           folder
        c) place the sundials.lib library into the new folder
        d) type (one line):

        link /out:test.exe 
           /LIBPATH:"C:\Program Files\Microsoft Visual Studio 8\VC\lib"
             cvbx.obj sundials.lib

        e) run "test.exe"



/Henning
