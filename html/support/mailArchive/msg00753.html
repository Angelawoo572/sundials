I agree with Adrian that this is not the place to discuss C vs. Fortran.

I'd only like to add that 3 of the current SUNDIALS solvers (CVODE, IDA, and KINSOL) provide a Fortran interface. Therefore, one can write Fortran code to be used with SUNDIALS without knowing any C. Of course, there's still the problem of providing the appropriate libraries at link time. For those using configure to build and install SUNDIALS, the next release will provide a script which can be used to obtain the required flags for linking to the SUNDIALS libraries.

--Radu
