Hi Jordan,

Converting a void* to int* without an explicit cast is legal in C, but not in C++...
So, from this perspective, this is a bug in the check_flag function in our examples.
We will make the fix in our examples, since -- with this exception -- both the libraries and example programs can be built using a C++ compiler.

Thank you for pointing this out,
--Radu
