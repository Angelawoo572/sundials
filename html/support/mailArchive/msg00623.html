Hi again,

I attached a small archive with the project file for creating a Sundials DLL (version 
2.2.0) with BCB6. Simply extract the directory "projects" into the sundials dir 
and open + compile the dllSundials.prj.

In the file 'sundials_dll.cpp' simply add dll-exported wrapper functions for 
all the functions you need to use from  Sundials.

To use the DLL, either dynamically load it via LoadLibrary() or link the 
dllSundials.lib file into your BCB6 project.
