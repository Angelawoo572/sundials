Radu,

As the author and maintainer of the parallel finite element micromagnetics package "magpar"

http://magnet.atp.tuwien.ac.at/scholz/magpar/

I would like to thank you, Alan Hindmarsh and all other contributors for your great work on SUNDIALS in general and PVODE in particular, which is one of the core libraries for my finite element micromagnetics model to integrate the equation of motion of the magnetization (Landau-Lifshitz-Gilbert equation).

With SUNDIALS version 2.0 a completely new API has been introduced and I will have to adapt my application "magpar" accordingly. However, this will take some more time and anyone who tries to compile and install "magpar" will need an older version of PVODE. Currently, I provide a link and a local copy of sundials on my server

http://magnet.atp.tuwien.ac.at/scholz/magpar/download/

which is not an ideal solution for you, because you cannot track these downloads.

Would it be possible to make older versions (<2.0) of the SUNDIALS suite (and possibly also all newer versions) available for download from

http://www.llnl.gov/CASC/sundials/download/download.html

or a similar page?

Recently, we had similar issues with PETSc and TAO
http://www-unix.mcs.anl.gov/petsc/petsc-as/
http://www-unix.mcs.anl.gov/tao/

which also affected the developers and users of deal.II (a FE library, which can use the parallel solvers of PETSc)
http://www.dealii.org/

Now both, PETSc and TAO, provide access to all versions:
http://www-unix.mcs.anl.gov/petsc/petsc-as/download/index.html
http://www-unix.mcs.anl.gov/tao/download/index.html


Thanks and kind regards,

Werner Scholz
