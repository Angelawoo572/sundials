Hi,

Over the past couple of weekends I've been playing around with the Google Maps API and thought it'd be interesting to see where all of you are :-) so I created a "sundials-users map" where you can each add your info (name, location, and an optional link to a photo).

The URL is http://www.llnl.gov/casc/sundials/sunmap/sunmap.html (beware that the Google Maps API key is case sensitive so only this link will work). Your browser must have JavaScript enabled and must accept cookies. You will "own" only one marker which you can delete if you want (from the table under the map). If you attempt to add another marker you'll be warned that this will replace the old one. I provided a short help page (link in top left corner).

I will not add links to this page from any other SUNDIALS web pages and I retain the 
right to remove markers that I do not "like" :-)

I'm looking forward to seeing this map populated!
