Michael Oevermann wrote: 
Hello all,

I am having problems to find the correct implementation of my DAE problem. Because I am
quite new to this field, I'll try to describe my problem here.

I want to solve a 1-d problem and I have n+1 grid points with indices 0,...,n. My dependent variables
are
y (mass-fractions),
u (velocity), and
p (pressure).

The pressure is spatially homogeneous!

For the mass-fractions I have normal ODE's

y'_i = f_i(y,u,p),  i=0,...,n

For the velocity I have an equation for the spatial derivative:

(du/dx)_i = p'  /  p + g_i(y)        (1)

with B.C.:

u_0 = u_L
u_n = u_R

(Note: du/dx does not depend on u itself and because p is spatially homogeneous I have only one pressure here!)

Given y_i and p, and p'  I can integrate directly:

=> u_i = u_0 + Sum_{k=1}^{i-1} (du/dx)_{k-1}  (2)

However, I have two B.C. (u_L and u_R) for this first order equation. Therefore, I am integrating only
to n-1 and not to n. The last equation (du/dx)_{n-1} I want to use for the pressure update:

u_R = u_{n-1}  + (du/dx)_{n-1} * dx = u_{n-1} + p'/ p * dx + g_{n-1}  dx

This can be re-arranged to an ODE for

p' = ( (u_R - u_{n-1})/dx  - g_{n-1} ) * p   (3)

In order to keep the Jacobian matrix banded, I introduce the artificial equation dp/dx = 0 and have an additioal
algebraic equation:

p_{i} = p_{i+1},   i=0,...,n-1    (4)

and from (3) I say

p' _{n} = ( (u_R - u_{n-1})/dx  - g_{n-1} ) * p_n  (5)

Now I have the following questions:

i) If I implement an algebraic equation (4) for p_i, i=0,...,n-1, do I have local values of p'_i as well?
(I need them to keep the system banded in (1) and (2))
ii) Somehow I don't get this system to work with IDA. Does anybody see if I make a conceptual mistake, e.g.
the system is not index 1 and cannot solved with IDA or anything else?
iii) in principle, u_{n-1} in (5) depends on ALL y_i, p_i, p'_i via the spatial integration (2). However, solving
for u as well, p'_{n} should depend on u_{n-1} directly and the Jacobian should be banded. Is that right?

Many thanks for your time to think about my problem!

Best regard

Michael
