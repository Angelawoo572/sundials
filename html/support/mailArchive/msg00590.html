The error messages are
        warning: dereferencing 'void *' pointer
        error: void value not ignored as it should be


void * are simply pointer dummies, you need to use type casting to tell the 
compiler specifically which variable you pass.

So for the call to CVODE you need to write:  (void 
*)(my_variable_array_or_whatever)
to cast your variable type into the dummy void * type

And inside your function you need to cast the pointer back: 
(my_variable_type)(f_data)
