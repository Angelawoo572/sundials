[sundials-users] Question about general paparell ode solver algorithms

From zxd@bu.edu
Date Fri, 4 Aug 2006 11:50:40 -0400 
Hi,

This is not sundials question but related to the parallel implementation of ODE solver algorithms. I think there are lots of expert here so I just throw out my question here and hope to get help.

To solve IVP problem A y' = f(t,y) where A is constant band-matrix. In rugge-kutta explicit method, y' is evaluated at every time step, and is very quick to solve the problem using LU factorization in serial mode. Most parallel implementation use newton iteration and
krylov space (GMRES) which in fact is much slower than the LU factorization method.

My question is: if y' is continuous, can we utilize some information
of y' at previous time (t-dt), to approximate y' at time t accurate enough, (that is, we approximate the problem with a delayed differential equation essentially).

If it can, I can solve A y' = f(t,y) in parallel using LU method (just use some boundary condition value at t-dt).

I wonder if there is some research on this problem, and any conclusion about that???

Many thanks
