Hi there,

I tested the make system with two compilers on the windows plattform and it works nicely (Intel compiler is next...)

MinGW32:
- Works perfectly, C alone, and also C and Fortran mix

Visual C++ Studio 2005 Professional:
- Works perfectly for the C build (project files are created, use "Visual C++ 2005" config)
- Hint: after the project files have been created, remove the CMakeLists.txt rules from your project files to avoid that modifications to the projects get overwritten next time you rebuild the solution
