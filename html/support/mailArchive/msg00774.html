Hello Ronan,

It sounds like you have not yet built a well-posed mathematical model.
For the "dependent variable" in question (call it V), you need to
decide whether V satisfies an ODE or is given by a separate explicit
equation of some sort.  It can't be both.  If it is given by a
separate equation, then it is not a dependent variable, and the ODE
system should be formulated with V(t) as a given function (or perhaps
computed as function of the other dependent variables), and not a
component of the y vector.

-Alan H
