Dear CW,

As you have noticed, the IBM XL C/C++ compilers use the GNU C and C++ header files and run-time libraries on Linux systems.

Try building SUNDIALS without any optimizations:

  CFLAGS="-g -O0"
  FFLAGS="-g -O0"

Note: -O0 is the letter O followed by the number zero.

--Aaron

--- Aaron Collier <collier6@llnl.gov> wrote:

Dear CW,

When using the IBM compilers you must make
certain
you are including the appropriate header files, so try the following:

   $ configure CC=xlc F77=xlf \
     CPPFLAGS="-I/usr/mill/include" \
     CFLAGS="-qarch=auto -qtune=auto" \
     FFLAGS="-qarch=auto -qtune=auto"

Note: According to the ANSI-C Standard, the
constant
DBL_MAX should be defined in float.h.

--Aaron
