Hi Alan and Andreas,

Up to now it is not possible to compile C or C++ Code with Delphi because there 
isn't a C++/C-Compiler in the package. 
This will come with the next major version of Delphi (begin next year). 
But Delphi and Borland C++ compilers using the same object format (omf). 
Therefore it should be possible to link precompiled object code together. 
You have to look carefully to your functions because Delphi and C++ have 
different calling conventions (see in help for modifier pascal and cdecl).
