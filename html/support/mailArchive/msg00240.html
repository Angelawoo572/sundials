Glad to hear that. You may also want to take a look at sundialsTB which provides MEX files to wrap around CVODES and KINSOL. Before downloading it, take a look at its documentation to see if it meets your needs. (http://www.llnl.gov/CASC/sundials/documentation/stb_guide/sundialsTB.html)

Being a Windows user and not having written C programs for a couple of years, I initially had a bit of trouble building the needed libraries.
After some hours of trial and error it turned out to be very easy.

I am still amazed that it can take hours to install SUNDIALS... Except for very particular hardware/software configurations -- which is not the case for usual Linux or Windows+cygwin -- it really takes me 5 min. to install SUNDIALS.
configure + make + make install worked every single time for me.

I must assume that many of the problems that people encounter have to be related to faulty installations of cygwin, mingw, etc.

Having said that, I'll mention that, for the next release (hopefully in about 1 month), we did make some more updates to the build system and that the documentation will describe in more details how to build SUNDIALS without using the configure script.
