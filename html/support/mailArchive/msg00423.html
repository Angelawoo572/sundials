Hi Aendey,

if nothing prevents you from using VC8 (Visual Studio 2005), I would recommend 
that you use that instead of VC6. The errors you mention are likely caused by 
older/obsolete c++ standard libs and shouldn't appear in newer versions of 
Visual Studio. However, if you want to use VC6 something to try out would be:

- create a project that only includes c files (don't link against sundials 
libraries but include the c files directly)
- if that works, try to separate SUNDIALS files step-by-step into a library 
and link against your program, perhaps you can find the problem with your 
project files

Andreas
