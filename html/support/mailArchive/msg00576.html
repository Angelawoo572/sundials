Dear All
I am attempting to solve a structure in the state space using IDA. Everything went fine until the program suddenly crashed. I found out that at some point in the iterations the function IDAStep is called but the nflag from the function IDANls is not IDA_SUCCESS so the program doesn't call the function IDATestError, therefore the variable err_k is not defined and when trying to call IDAHandleNFlag, it crashes because err_k is not defined.
As a "solution" I uncommented the if test before the call to IDATestError so the variable would be defined at all times.
I still have to do some more checking on my results but they seem very reasonable.
 
 I am not very familiar with the variables used along the program and not too familiar with all the mathematical procedures either so there probably is a better way to solve this problem so if you can help me with some hints, I will be grateful.
