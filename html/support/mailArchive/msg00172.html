Dear Andreas Nicolai,

Regarding the use of a different math library, the method you mentioned is probably the easiest, but you'll also need to modify shared/source/Makefile after running the configure script. You'll need to make certain that the LIBS variable is set to the following:

  LIBS = -limf -lm

Note: The order in which the libraries are linked is important.

This is necessary because there's currently no way for the user to specify a library with "higher precedence" than the standard math library.

I'd also recommend that you specify the following argument when running the configure script:

  CPPFLAGS=-no-gcc

By default, the Intel C/C++ compiler sets certain environment variables indicating that it's the GNU C compiler which confuses the configure script.

Concerning the use of a different compiler, I'm not sure I understand your request. Defining a different compiler can easily be accomplished by setting the CC and/or F77 environment variable(s).

--Aaron
