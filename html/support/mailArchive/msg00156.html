On Sat, 22 Oct 2005, Vinayak Kumar wrote:

> Hi,
>
> I want to ask just one simple Question. I was trying to solve a set of
> equaitons using IDA. The equations are given as :
>
> [1] (1e-3) * (y6)' - y3 + y2 = 0
> [2] -(1e-6) * (y3)' - y5 = 0
> [3] y2 - y1 - (100) * y4 = 0
> [4] y4 - y5 = 0
> [5] y5 - y6 = 0
> [6] y1 - 5.0 = 0
>
> These are equations of a simple RLC netwrok. I am using IDADense() method
> and User Defined Jacobian function with correct initial values of the
> variables.
>
> Will the solver be able to solve it? Actually I am getting some error
> messages. Please tell me. If not then may you please provide me some
> explanation?
>
> Thanks and Regards,
> Vinayak


Hello Vinayak,

This system looks well-posed to me, with index = 1 (in DAE terminology).
If you give IDA consistent initial values for all yi (with y1, y2, y4,
and y5 all being determined from the given y3 and y6), it should work.
If it fails, try calling the initial condition calculation routine
IDACalcIC before calling IDASolve.  Also, double check your Jacobian
routine, and try the internal difference-quotient Jacobian routine
option instead, as a check.

If you still have failures, you might want to reformulate your system
as one with a minimal number of variables.  In this case, you can
reduce it to a system of 2 ODEs, in y3 and y6.

-Alan
