Hi Robert,

All CVODE examples that use the f_data, use it to pass an entire structure, not only a scalar. It should be no different passing an array of realtype variables. Something like:

void main(...)
{
realtype *my_data;
...
my_data = (realtype *)malloc(10*sizeof(realtype));
...
cvode_mem = CVodeCreate(...);
flag = CVodeSetFdata(cvode_mem, my_data);
flag = CVodeMalloc(cvode_mem, my_f, ...);
...
}

static int my_f(realtype t, N_Vector u, N_Vector udot, void *f_data)
{
realtype *my_data;
...
my_data = (realtype *)f_data;
...
return(0);
}

--Radu
