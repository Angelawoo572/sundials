We built CVODE using cygwin aswell but it did require some knowledge of how 
make scripts are used under Unix. This is not something windows developers may 
be aware of (just as Unix developers may not be aware of how to use Visual

VS project files with the distribution as it makes installation for windows 
users a trivial exercise.
