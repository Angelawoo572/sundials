Herbert Sauro wrote: 
We built CVODE using cygwin aswell but it did require some knowledge of how 
make scripts are used under Unix. This is not something windows developers may 
be aware of (just as Unix developers may not be aware of how to use Visual 
Studio). However, if it were at all possible I would recommend providing some 
VS project files with the distribution as it makes installation for windows 
users a trivial exercise.

Herbert Sauro


As I must have already mentioned in a previous message, we've updated the documentation for the next release to include more details on building SUNDIALS without the *NIX configure script. We list there what files are needed, where they are located, provide a sample sundials_config.h header file, etc.

Also, the new website will have a FAQ section which already contains several entries on installation questions. I think that would be the best place to provide project files for VC, Borland C++, etc. None of us here use any of these so one of you would have to provide them. Please email them to me and I will add them to the website.

--Radu
