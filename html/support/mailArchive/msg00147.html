It's all described in the IDA user guide (the "Example Programs" document 
refers you to that!):
http://www.llnl.gov/CASC/sundials/documentation/ida_guide/node6.html#SECTION00653000000000000000

cj is the \alpha in Eq. (3.5):
http://www.llnl.gov/CASC/sundials/documentation/ida_guide/node4.html#e:DAE_Jacobian

cj is an input to the Jacobian function. This function computes
J = F_y + cj*F_y'
for given t, y, y', and cj
