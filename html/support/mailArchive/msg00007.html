Hello Jordan,

Yes, SUNDIALS can be used under Windows. The easiest way is to use cygwin and use the configure script as under any *NIX environment. I have tried this and there are no special instructions needed beside those in the user guide. I have also heard from users that built it using Visual C++. In this case however, you must first manually create the required header file sundials_config.h (using the template sundials/config/sundials_config.in). This header file is otherwise automatically generated by the configure script.

I had hoped that this mailing list will receive a bit more traffic, particularly users helping other users (unfortunately, many users still prefer to email us directly). I have thought about archiving previous messages, but there are not that many...

For those of you that have built SUNDIALS under Windows, this is a good 
opportunity to share your experiences.

Thanks,
