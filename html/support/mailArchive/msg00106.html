I don't, but I want to warn anyone using that C version of LSODA.  It is
based on a version from 1987.  There have been MANY corrections and
improvements to the Fortran LSODA since then.  The current version is
available from Netlib/odepack.  So a fair comparison should use either
the current Fortran LSODA or a f2c translation of it.
