Hi James,

to give you a third alternative valid for C++:

if (typeid(realtype) == typeid(double)) std::cout << "double found\n";

Using sizeof does give you the size in bytes, but I think the size of a
datatype is not guaranteed by C/C++, so this is not platform
independent. It may happen that two datatypes need the same amount of
memory.

Using the limits library is a workaround that should work.

Also think about whether you really need to do a runtime check. In most
of the cases where I needed type information there was a better way to
solve the problem.

Hope this helps,

Kai
