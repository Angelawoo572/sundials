Hello KP,

> When I set up just a 2-cell model with cells of the same kind, I find
> both the cells give me the exact same results though they have been
> initialised differently!

This statement by itself means that you are not using CVODE correctly,
somehow.  If cells 1 and 2 really have different initial values, and
you look at the solution soon enough (before both have damped to a
common steady state), you will see different values for their solutions.
You probably have an indexing bug of some kind.

There is no special advice for setting up a multiple cell model. If
there are N1 dependent variables for one cell, and there M cells, then
initialize CVODE for a problem of size N = M*N1.  Of course the cells
will have "different state vectors at each time step" in general.

-Alan H
