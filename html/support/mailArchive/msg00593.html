Quoting Radu Serban <radu@llnl.gov>:

Hi Xuedong,

This tends to be a problem-specific issue, so we'll probably need a bit more information on yours. However, there are several general
techniques that may work, such as rescaling your states and/or equations or reformulating your mathematical model (e.g. switch from an ODE
to a DAE description). Hopefully, I'll be able to be more specific once you give a more detailed description of your problem: what is the
physical phenomena you try to model, how large is the resulting DE, what causes the ill-conditioned Jacobian, etc.

In the meantime, since you mention having tried one of the SPILS (scaled preconditioned iterative linear solvers), I'd like to make sure
that you are building a preconditioner for the right matrix (I realize that the documentation is not very clear on this point - we'll have
to add some clarifications to it). Consider F(y,y',t) = A y' - By + x(t) = 0. You have the following two options:
1) Transform it to an explicit first-order ODE (and use CVODE)
y' = A^{-1} B y - A^{-1} x(t)
As Alan mentioned, this assumes that you can efficiently compute A^{-1} every time the RHS is evaluated. However, if A is ill-conditioned,
this may be out of the question.
2) Treat it as an implicit first-order ODE (and use IDA)

If using CVODE with Newton or IDA and if you use a SPILS, the preconditioner matrix P must be an approximation to the resulting Newton
iteration matrix which, for the above two cases, is
1) M = I - gamma * J = I - gamma * df/dy, where f(t) = A^{-1} ( By-x(t) ) and gamma is a scalar proportional to the integration step size h.
Therefore (ignoring the constant in gamma), in your situation, P must approximate
P ~ I - h A^{-1} B = A^{-1} ( A - h B )
2) M = dF/dy + cj dF/dy', where F(t) = Ay'-By+x and cj is a scalar proportional to 1/h. In your situation:
P ~ -B + 1/h A = 1/h ( A - h B )

Are you preconditioning the right matrix? Is P also ill-conditioned?

A couple other questions:
- are both A and B ill-conditioned?
- is your problem small enough to simply try the banded linear solver?
- you mentioned using an LU preconditioner. Do you mean an incomplete LU? Otherwise, this would effectively be the same as using the dense
linear solver.

--Radu
