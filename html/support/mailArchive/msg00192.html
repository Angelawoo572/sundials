N_Vector & solver module unit test suite?

From Kevin Long <krlong@sandia.gov>
Date Fri, 09 Dec 2005 16:44:04 -0800 
Hi,

Does there exist an abstract unit test suite for N_Vector and solver modules? 
Such a thing would be helpful for people trying to wrap vector and solver 
libraries in the SUNDIALS interfaces. Because N_Vector and the linear solver 
modules are representation independent, a single test suite could be used for 
all N_Vector/solver implementations. As one who is fanatical about testing as 
I develop, I'd feel safer if I could know that my vector and solver wrappers 
are behaving as expected by SUNDIALS before going on to solving DAEs.

I can (and do) test all the vector operations required by SUNDIALS within the 
context of my own vector library, but it would be handy to test them through 
the SUNDIALS interface as well.

Thanks,

- kevin
