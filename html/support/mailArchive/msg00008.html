Is it possible to compile and use CVODE in a windows environment? If so, are there any instructions available for how to do so? 


Hello Jordan,

Yes, SUNDIALS can be used under Windows. The easiest way is to use cygwin and use the configure script as under any *NIX environment. I have tried this and there are no special instructions needed beside those in the user guide.
...
For those of you that have built SUNDIALS under Windows, this is a good opportunity to share your experiences.


Yes, SUNDIALS works fine under Windows/Cygwin. I have compiled
the latest SUNDIALS version on Windows 2000 with GCC 3.4.3
without any problems.

Providing this binary for download makes no sense because
you need the cygwin dll's for running SUNDIALS.

I would propose the following steps:

1. Install the Cygwin software (http://www.cygwin.com/)
   - download the setup.exe program
   - start setup.exe (as privileged user)
   - I prefere an empty partition for Cygwin like D:\
   - complete installation (it takes a while) - you need
     access to Internet for downloading all the Cygwin packages

2. Prepare a shell environment
   - add D:\bin to the PATH variable (Settings - System -
     Advanced - Environment ...
   - start D:\bin\rxvt.exe (the shell)

3. Download the SUNDIALS source package and unpack it
   - mkdir /tmp
   - cd /tmp
   - tar xzvf sundials.tar.gz   (or similar)
   - cd sundials

4. Read the installation instructions in the file INSTALL
   - in many cases three commands are important:
     ./configure
     make
     make install


Maybe, this helps.
