Hi Yana,

I have not tried to install the currently released sundialsTB under Windows, but I know that several users on this mailing list have successfully done so and maybe they could give you some directions. To do it under cygwin, you may want to take a look at gnumex.

However, for the next SUNDIALS release, precisely to make the sundialsTB installation less platform/OS dependent, I changed it so that it is done from within Matlab, by running a Matlab script. Moreover, the new installation procedure does not assume that the SUNDIALS libraries are precompiled, but rather compiles the necessary SUNDIALS sources with Matlab's mex at the same time as the interface functions.

I hope that we will have the new SUNDIALS release before the end of the month. However, if you cannot wait for the new release, I will try to retrofit the currently released sundialsTB with the Matlab installation script and post it to the mailing list.
