Just an idea. Feel free to ignore if you guys dont like it. I consider
providing sundials for free itself is of great help. But giving CVS
read access would be of even more help.

Is there any good reason for doing this (ie not giving read access to
the CVS repository) which I missed?

As you probably know, you can give read access to others for the cvs
repository and restrict the write access for whomever you want. I
think that way people can have access to the latest source and can
even send you guys some patches/enhancements to make sundials a much
better software.

One reason that I could think for doing this is the extra time it
takes to support CVS version as well. But you can always say that CVS
version is not supported and it is up to the end user whether to use
unsupported CVS or supported stable releases.

just my 2 cents
