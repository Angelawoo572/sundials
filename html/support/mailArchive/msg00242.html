I am still amazed that it can take hours to install SUNDIALS... Except
for very particular hardware/software configurations -- which is not the
case for usual Linux or Windows+cygwin -- it really takes me 5 min. to
install SUNDIALS.

related to faulty installations of cygwin, mingw, etc.

that the documentation will describe in more details how to build
