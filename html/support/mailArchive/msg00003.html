IER=-6 corresponds to KIN_MAXITER_REACHED, so the solver reached the maximum number of nonlinear iterations allowed (default 200, can be changed through KINSetNumMaxIters() or specified in IOPT(2) when using FKINSOL).

The ** for the linear solver return value (IOPT(15)) is probably due to an incorrect format descriptor (see declaration of IOPT).

Hope this helps,
--Radu
