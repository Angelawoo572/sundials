The {\ida} distribution contains, in the \id{sundials/ida/examples\_ser}
directory, the following four serial examples (using the {\nvecs} module):
\begin{itemize}
\item \id{iheatsb}
\item \id{iheatsk}
\item \id{irobx}
\item \id{iwebsb}
\end{itemize}

\noindent In the \id{sundials/ida/examples\_par} directory, the {\ida} 
distribution contains the following four parallel examples 
(using the {\nvecp} module):
\begin{itemize}
\item \id{iheatbbd}
\item \id{iheatpk}
\item \id{iwebbbd}
\item \id{iwebpk}
\end{itemize}
